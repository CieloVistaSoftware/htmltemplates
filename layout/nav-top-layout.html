Main Content Area

Welcome to the Nav Top Layout template. This layout places the navigation bar at the very top of the page, followed by a header section, the main content area, and finally a footer.

Once you've chosen this layout, you can begin customizing the content by dragging elements into this main section.

Navigation First

This layout prioritizes navigation by placing it at the top of the page, ensuring users can immediately access important links.

The compact navigation bar includes essential links and search functionality, conserving vertical space while maintaining accessibility.

This approach is ideal for content-rich websites where easy navigation is crucial.

Prominent Header

Below the navigation, a spacious header area provides an opportunity to showcase your most important messaging or call to action.

This section is perfect for welcome messages, announcements, promotions, or key value propositions.

Content Focus

The large main content area allows for extensive information presentation with clear visual hierarchy.

This layout works well for blogs, information portals, documentation sites, and any content that requires scrolling.

  • Works well on various screen sizes
  • Provides balanced visual weight
